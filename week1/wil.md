Github에 어떤 게시물을 올리는 것만 가능하다고 생각했지, 터미널 등과 연결해 사용할 수 있다는 점을 알았다. 홈페이지를 내가 직접 만드는 것은 항상 막연하다고만 생각했고, 여러 코드를 사용해 간단하지만 내 웹사이트를 만들었다는 점에서 흥미로웠다. vercel 이라는 새로운 홈페이지와 html의 주요 코드 등을 배웠으며 Git을 사용해 프로그램 명령어를 입력하고 실습을 하는 동안 처음 접하는 것이라 따라가기가 조금 벅차다고 느껴지는 부분이 있었다. 하지만 시간 내 완성한 결과물을 보니 대단한 것은 아니었지만 뿌듯했던 듯하다.
