# 새로 배운 명령어
## 1. git 명령어
- `pwd`: 현재 폴더의 위치를 알려준다. 
- `git clone`: 이를 적은 후 URL을 적으면 github의 코드를 컴퓨터로 가져올 수 있다. 
- `ls`: 이 명령어는 현재 위치에 어떤 파일이 있는지 알 수 있다.
- `cd`: 이 명령어를 사용하면 폴더 위치를 변경할 수 있다.

 ## 3. HTML 관련 명령어
 - `a href` 명령어를 사용하면 다른 페이지로 이동할 수 있게 된다.
 - `HyperText`를 사용하면 저번에 만들었던 웹사이트에서 다른 페이지로 이동할 수 있다.
